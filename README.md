# MFJ Travel & Transfers

Luxury private transfer website for Galway, Connemara, Shannon Airport, Dublin Airport and Knock Airport.

## Features
- Next.js 15 App Router
- TypeScript and Tailwind CSS
- Black and gold luxury responsive design
- Dedicated pages for Home, About, Services, Fleet, Booking, FAQ and Contact
- Sticky navigation, animated hero, maps, reviews and WhatsApp CTA

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push the repository to GitHub.
2. Import it into Vercel.
3. Vercel will detect the Next.js project automatically.
